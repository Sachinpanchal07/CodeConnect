# CodeConnect
A full-stack MERN + Socket.io app where users can connect, chat, send requests, and make payments with Razorpay.

## Features
- OTP-based registration and JWT authentication
- Real-time chat using Socket.io
- Send and accept connection requests
- Scheduled notifications using Cron Jobs
- Payments via Razorpay
- Responsive UI with Tailwind CSS

## Tech Stack
**Frontend:** React.js, Vite, Tailwind CSS  
**Backend:** Node.js, Express.js, MongoDB, Socket.io  
**Other Tools:** JWT, Razorpay API, Cron Jobs, Render, Vercel

## Installation
cd codeconnect

#Backend Setup

cd backend;
npm install;
npm run dev;

#Frontend Setup

cd frontend;
npm install;
npm run dev;
